Scrollbars

Firefox has the ability to put custom colors on your scrollbars! Don't forget to put candy-scrollbar on your <html> element if you'd like to opt-in to this feature. You can also put it on only specific elements that you'd like to opt-in to this behavior. Hopefully in the future more browsers besides Firefox will implement this feature.

You can also use the nonstandard WebKit scrollbar properties to color scrollbars if you'd like. This is not included in candy.css itself. 

::-webkit-scrollbar {
  width: 1rem;
  height: 1rem;
  background-color: var(--candy-color-background);
}

::-webkit-scrollbar:vertical {
  border-left: 1px solid var(--candy-color-shadow);
}

::-webkit-scrollbar:horizontal {
  border-top: 1px solid var(--candy-color-shadow);
}

::-webkit-scrollbar-track {
  padding: 2px;
}

::-webkit-scrollbar-corner {
  background-color: var(--candy-color-shadow);
}

::-webkit-scrollbar-thumb {
  background: var(--candy-color-text);
  background-clip: content-box;
  box-shadow: inset 0 0 0 3px var(--candy-color-background);
}

::-webkit-scrollbar-thumb:vertical {
  border-left: 1px solid var(--candy-color-shadow);
  min-height: 1rem;
}

::-webkit-scrollbar-thumb:horizontal {
  border-top: 1px solid var(--candy-color-shadow);
  min-width: 1rem;
}

Dark Mode

You can use the prefers-color-scheme: dark CSS media query to set the --candy-color-* variables, allowing your visitors to automatically get dark mode with a single block of CSS 

.candy-auto,
.candy-root {
  --candy-color-background: #fff;
  --candy-color-text: #333;
  --candy-color-shadow: #666;
  --candy-color-accent: #c00;
}

@media (prefers-color-scheme: dark) {
  .candy-auto,
  .candy-root {
    --candy-color-background: #333;
    --candy-color-text: #ccc;
    --candy-color-shadow: #666;
    --candy-color-accent: #0dd;
  }
}

Auto Mode

The suggested way to use candy.css is exclusively by adding CSS classes to elements you want to style. But sometimes your HTML is constrained to be entirely class free.

In order to use auto mode, put the class candy-auto on whichever tag you want class free markup inside of (e.g. on the main element of your blog post, or even the entire document if you want to work quicker, or use Markdown or something).
