candy-css

Introduction

candy.css is a small CSS library for making retro-inspired web pages using four colors. Its color palette is completely customizable using CSS Custom Properties.

More Detail

You can use candy-css with basic form fields and other elements. candy-css requires the use of CSS classes, so it won't clash with your existing CSS. candy-css also uses relatively low specificity for each rule, making it much less likely to override your existing CSS.

candy-css is about 18 KB of unminified CSS and compresses to about 2.6 KB via gzip compression.

Standard Installation

Download candy.css and use a link tag: 

<link rel="stylesheet" href="candy.css" />

Webpack Installation

If you're using webpack with css-loader, you can install it through npm: 

npm install candy-css

Then load the CSS through your JavaScript:

import "candy-css";
